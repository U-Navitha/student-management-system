# Banking Management System

## Overview
A simple Java-based Banking Management System that allows users to create accounts, deposit money, withdraw money, and view account details.

## Features
- Create Account
- Deposit Money
- Withdraw Money
- Display Account
- Display All Accounts

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- Scanner Class

## How to Run

1. Compile the program:
javac Main.java

2. Run the program:
java Main
